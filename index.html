🔑 雲端與金鑰設定

💡 如何取得 Gemini API 金鑰？ ▼
  1. 前往 Google AI Studio 並登入您的 Google 帳號。
  2. 點擊畫面左側選單的 Get API key。
  3. 點擊畫面中的 Create API key 按鈕。
  4. 選擇 Create API key in new project 即可免費生成。
  5. 複製以 AIza 開頭的長串金鑰，並貼入上方的輸入框中儲存即可。
🔥💡 如何部署並取得 GAS 網址？ (含同步程式碼) ▼
  1. ⚠️ 重要更新提醒：若您曾部署過舊版，請務必重新複製下方完整程式碼覆蓋舊版，並「建立新的部署作業」，才能完全修復置左對齊及中文化問題！
  1. 建立一個新的 Google 試算表，點選頂端選單的「擴充功能」>「Apps Script」。
  2. 將下方黑底白字的程式碼完整複製，並完全取代編輯器內原本的所有內容。
  3. 修改程式碼第一行的 ACCESS_PASSWORD，設定您專屬的同步密碼。
  4. 點選右上角的藍色按鈕「部署」>「新增部署作業」。
  5. 點選左側「⚙️」齒輪圖示，選擇「網頁應用程式 (Web App)」。
  6. 設定「執行身分」為「我」，「誰可以存取」選擇「所有人」，然後點擊部署。
  7. 出現授權提示時，完成 Google 帳號授權 (若遇安全警告請點選 左下角進階 > 前往)。
  8. 成功後，複製畫面上的「網頁應用程式網址」，貼入上方欄位即可。
